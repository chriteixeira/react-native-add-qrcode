# react-native-add-qrcode

Add a QRCode to an image

## Installation

```sh
npm install react-native-add-qrcode
```

## Usage

```js
import AddQrcode from "react-native-add-qrcode";

// ...

const result = await AddQrcode.multiply(3, 7);
```
